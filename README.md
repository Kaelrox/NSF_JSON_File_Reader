This is the software solution to implement our workfow for the convergence research FYAP investigative project
FSU FYAP project. The project revolves around convergence, which in this context denotes the concept of multiple fields of science working in tandem on a single project.

This source code is to implement the workflow we discussed. On a macro level the  workflow consist of  3 tasks
1: Ingesting the data via JSON files  downloaded  from the NSF
2:Pushing the data  to a SQL and NOSQL  database
3: Datamining and output
